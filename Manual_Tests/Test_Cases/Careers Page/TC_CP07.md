Test Case ID : TC_CP7.md

Prerequisites : none

Title : Career search function validation - filtering validation.

Description : Validate that the careers page search function's filtering works as intended.

Steps :
1. Open https://www.mcdonalds.com/de/de-de.html
2. Scroll to the bottom of the page (Home Page)
3. Press on Careers (Karriere) link. / Alternatively: Open https://karriere.mcdonalds.de/ directly.
4. Choose a job field ("Bereich")
5. Choose an area of expertise ("IT", "Marketing" etc.)
6. Choose a type of employement ("Vollzeit", "Teilzeit" etc.)
7. Choose a job level ("Ausbildung", "Direkteinsteig" etc.)


Expected result : Job openings are shown correctly based on used filters.

Pass / Fail : Pass

Title : Career search function validation - Without name/job description input.

Description : Validate that the careers page search function works as intended with limited data provided.

Steps :
1. Open https://www.mcdonalds.com/de/de-de.html
2. Scroll to the bottom of the page (Home Page)
3. Press on Careers (Karriere) link. / Alternatively: Open https://karriere.mcdonalds.de/ directly.
4. Leave Position field empty. (DO NOT INPUT ANY DATA)
5. Input valid Location ("52062 Aachen)
6. Choose "10 Km" from the distance drop-down menu.
7. Click Search ("Suche") or press enter.

Expected result : All the available job oppenings are displayed in the pre-required region.
